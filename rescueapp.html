Rescue Alert Application

This was a project we started back in June 2022 when we have the class Software Engineering I. I was the lead developer for this application with the help of my teamates we were able to create an application for the purpose of Road Accidents.

This application, we have created a single button that if it was click, the location and message of the victim will be sent to the pre-saved contacts of the application.

We've worked on this system application for weeks together with Noven Reyes, and Khalil Desamparado in order to achieve the desired result.

Technologies:
  • - Flutter
  • - Hive
  • - Dart
  • - Firebase
